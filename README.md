# React To-Do List Application.

# Project Description.
-> This is a simple To-Do List application built using React.
-> It allows users to manage their daily routine tasks efficiently dy adding, deleting, editing and marking tasks as completed or not.
-> clink on that task to check whether task is completed or not, it only toggles the task if it is completed or not. 

# Commands to run the project.
-> install node-modules using "npm install"
-> run the project using "npm run dev"

# Features Added.
-> Adding new tasks
-> Deleting tasks
-> Mark tasks as completed
-> Editing existing tasks
-> Simple and user friendly UI

# Technologies Used
-> React
->Vite
->JavaScript
->HTML
->CSS
